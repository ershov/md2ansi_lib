    * ## Ticket: Section 4: `_m2a_build_context` + placeholder rewrite {#8}

          status: done
          created: 2026-05-18 02:15:18 UTC
          updated: 2026-05-18 02:27:05 UTC

      Per design §5.3:
      - Define `_M2A_PLACEHOLDER_RE = re.compile(r"\(\?P([<=])\*(\w*)>")`.
      - Implement `_m2a_build_context(rules)`:
        1. For each rule rewrite `(?P<*name>...)` and `(?P=*name)` to `(?P<{rulename}_name>...)` / `(?P={rulename}_name)`. Empty suffix becomes `inner`.
        2. Wrap each rewritten pattern with `(?P<{rulename}>{rewritten})`.
        3. Join alternatives with `|`, compile under `re.VERBOSE | re.MULTILINE | re.DOTALL`.
        4. Return `M2A_Context(compiled=..., rules=tuple(rules))`.

      Verify on a one-rule fixture, e.g. `[("bold", r"\*(?P<*>[^*]+)\*", "1", None)]`: compiled regex matches `*x*`; `m.group("bold")` and `m.group("bold_inner")` resolve.
